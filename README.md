# SVCJ Factor Engine

A high-performance, pre-compiled engine for generating time-series SVCJ (Stochastic Volatility with Correlated Jumps) risk factors from financial data. This package is optimized for speed and is ideal for large-scale factor discovery and machine learning applications in quantitative finance.

The core solver is written in C and wrapped with Cython to provide a simple, high-level Python interface.
